6. RESPONSIVE BREAKPOINTS

@media (max-width: 767px)
Single column, stacked navigation, hidden sidebars
Tablet: 768px - 1023px
@media (min-width: 768px) and (max-width: 1023px)
Two columns, collapsible sidebar, flexible main content
Desktop: 1024px - 1439px
@media (min-width: 1024px)
Full three-column layout, all panels visible
Wide Desktop: 1440px+
@media (min-width: 1440px)
Centered max-width container, increased spacing
7. IMPLEMENTATION NOTES FOR CLAUDE CODE
Component Hierarchy: Each wireframe section represents a React component. Use the component names specified in the tables when creating files.
Styling Application: Apply your 80s retro aesthetic as a theme layer on top of this structure. Consider: neon borders, ASCII art decorations, pixel fonts, CRT screen effects, glitch animations, and synthwave color gradients.
Data Flow: Components should receive data via props from your existing backend (posts, profiles, auth state). Use your established Zustand stores for state management.
Component Category Retro Styling Suggestions
Buttons Pixel borders, neon glow on hover, ASCII art icons, retro sound effects
Cards/Panels CRT scanlines, phosphor glow, terminal-style borders using box-drawing characters
Typography Monospace fonts, green/amber terminal colors, blinking cursors, ASCII headers
Forms/Inputs Terminal prompts, command-line aesthetics, matrix rain backgrounds
Animations Glitch effects, scan line movements, pixel transitions, typing animations
Colors Neon pink (#FF10F0), electric blue (#00FFFF), laser green (#39FF14), purple (#9D00FF)
